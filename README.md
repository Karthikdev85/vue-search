# Vue Search
Fetch high-quality images from the Unsplash API
![alt text](<Screenshot 2024-11-28 143832.png>)

# Setup for local development
1. git clone https://github.com/{your-username}/vue-search.git
2. cd vue-search-component
3. npm install
4. npm run dev
